# VMKS_Speaker
project – Bluetooth speaker

ESP32 Bluetooth колонка
Описание

Проектът реализира проста Bluetooth аудио колонка, базирана на микроконтролер ESP32.
ESP32 приема аудио сигнал чрез Bluetooth връзка и го подава към външен усилвателен модул, свързан на breadboard платка.

Развойна платка ESP32

Breadboard платка

Джъмпер проводници

Външен говорител
